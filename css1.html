• css语法

    1. css主要由两部分构成：选择器和以{}包裹的声明
    2. 每条声明由属性和其值组成，多个声明用“；”隔开
    3. 选择器主要用class选择器，前要加“.”，id选择器前加“#”

  • 颜色，尺寸，对齐

    1. 颜色可以用其名称或者rgb(xxx,xxx,xxx,每种颜色用三个字节表示 )
      2. height和width设置元素内容占据的尺寸
    2. 对于元素中的文本，可以简单的设置text-align属性为left, center, right，缺省为左对齐

  • 盒子模型

    1. Content 盒子的内容，如文本、图片等
    2. Padding 填充，也叫内边距，即内容和边框之间的区域
    3. Border 边框，默认不显示
    4. Margin 外边距，边框以外与其它元素的区域

  • 边框与边距

    1. 无论边框、内边距还是外边距，它们都有上下左右四个方向
    2. 简写时的顺序为上，右，下，左，或记为顺时针方向

  • 定位

    1. static 静止：默认定位方式，元素按从上往下从左往右排列
    2. relative 相对：把元素相对于它静止时的位置进行移动
    3. fixed 固定：将使得元素在浏览器的屏幕可见区域内固定不动
    4. absolute 绝对：将使元素相对于其最近设置了定位属性（非static）
      的父元素进行偏移 如果该元素的所有父元素都没有设置定位属性，那么就相对于body这个父元素

  • 溢出

    1. overflow属性来处理这些溢出的部分。
    2. visible 默认值，溢出部分不被裁剪，在区域外面显示
      hidden 裁剪溢出部分且不可见
      scroll 裁剪溢出部分，但提供上下和左右滚动条供显示
      auto 裁剪溢出部分，视情况提供滚动条

  • 浮动

    1. 可以设置float属性让某元素水平方向上向左或右进行移动,其周围的元素也会重新排列
    2. 如果希望浮动元素后面的元素在其下方显示，可使用clear: both样式来进行清除。

  • 不透明度

    1. opacity对任何元素（不过常用于图片）设置不透明度,
      值在[0.0～1.0]之间，值越低，透明度越高

  • 组合选择器

    1. 后代选择器:以空格作为分隔，如：.haha p 代表在class=haha内的所有p元素
    2. 子代选择器(也称直接后代选择器)：以>作为分隔，如：.haha > p
      代表在有.haha类的元素内的直接p元素。

  • 伪类和伪元素

    1. 伪类（pseudo-class）或伪元素（pseudo-element）用于定义元素的某种特定的状态或位置等
    2. 使用方法：选择器后使用“:"号，再跟上某个伪类/伪元素
